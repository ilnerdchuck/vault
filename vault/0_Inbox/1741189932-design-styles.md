- Custom
    - + High Performance
    - - high cost
    - - High Design time
- Semi-Custom
    - Cell Based
        - STD-CELL ASIC
            - Library of standard Cells
            - Matrix where the standard cells are placed
                - All cell have the same height but can have variable width
    - Array Based
        - FPGA
            - + Flexibility 
            - + Low Design Cost
            - - Performance 
            - - Power
